<!-- Restraurand Agent Frontend website idea -->

A restaurant website built for a specific restaurant named "Hong China" in the city of Nairobi, Kenya.
The main features are:
1. Agent chatbot with customers
2. Order management

The website should be built with the following technologies:
1. React
2. Next.js
3. Tailwind CSS
4. TypeScript
5. Supabase

The UI of website should be built with the following design:
1. Home page with restaurant banner, logo, and a short description of the restaurant. It will sections -
    1. Menu section
    - Best items
    - New items
    - Popular items
    - Special offers section
    2. Chatbot widget at the bottom right of the page
    - Chatbot icon with a chat bubble icon
    - Chatbot chat window
    - Input
    3. Customers orders section
    - Order tables with status, items, prsice, type etc. 
        - current orders
        - past orders etc.
    4. Delivery tracking section
    - For delivery order tracking - status(puicked, in transit, delivered), time needed, items, price,
    - Delivery person tracking on map    



    
    ## Beautiful Styling Theme for Hong China Restaurant Website
    
    ### Color Palette
    - **Primary Colors:**
      - Deep Red (#DC2626) - Traditional Chinese red for prosperity and good fortune
      - Gold (#F59E0B) - Luxury and elegance
      - Dark Green (#059669) - Fresh ingredients and nature
      - Cream (#FEF3C7) - Warm and inviting background
    
    - **Secondary Colors:**
      - Charcoal (#374151) - Professional text and borders
      - Light Gray (#F3F4F6) - Subtle backgrounds
      - White (#FFFFFF) - Clean spaces
      - Orange (#EA580C) - Call-to-action buttons
    
    ### Typography
    - **Headings:** 'Playfair Display' - Elegant serif font for restaurant name and section titles
    - **Body Text:** 'Inter' - Clean, readable sans-serif for content
    - **Chinese Text:** 'Noto Sans SC' - Proper Chinese character support
    
    ### Design Elements
    - **Borders & Shadows:**
      - Subtle rounded corners (8px-16px)
      - Soft box shadows with 0 4px 6px rgba(0, 0, 0, 0.1)
      - Gold accent borders for premium sections
    
    - **Icons & Graphics:**
      - Traditional Chinese patterns as decorative elements
      - Food icons with warm, appetizing colors
      - Minimalist chat bubble design for chatbot
    
    ### Layout Structure
    - **Header:** Fixed navigation with restaurant logo and menu
    - **Hero Section:** Large banner with restaurant image and tagline
    - **Content Areas:** Card-based layout with consistent spacing
    - **Chatbot:** Floating widget with smooth animations
    
    ### Interactive Elements
    - **Buttons:** Gradient backgrounds with hover effects
    - **Cards:** Subtle hover animations and transitions
    - **Forms:** Clean input fields with focus states
    - **Chatbot:** Smooth slide-in animation and typing indicators
    
    ### Responsive Design
    - Mobile-first approach
    - Flexible grid system
    - Touch-friendly interface elements
    - Optimized for all screen sizes
    
    ### Accessibility
    - High contrast ratios
    - Clear focus indicators
    - Screen reader friendly
    - Keyboard navigation support
